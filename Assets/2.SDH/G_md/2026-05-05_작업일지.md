## 신동혁 (SDH) 작업 내역

### 완성된 시스템

#### 1. 적 생성 시스템 — SpawnPoint.cs
**경로:** `Assets/2.SDH/Scripts/SpawnPoint.cs`

- Dictionary 기반 오브젝트 풀링 (prefab별 Queue 관리)
- 5방향 독립 스폰 인터벌 (상단 하강 / 좌→우 / 우→좌 / 대각선 2방향)
- 스폰 시 EnemyController에 방향·스포너 주입 후 onDie 대리자 등록

```csharp
ec.onDie = ItemManager3.Instance.CreateItem;  // 적 사망 → 아이템 드랍 연결
```

**포인트 배치:**
- Point 0~4: 위에서 아래 (Vector3.down)
- Point 5~6: 오른쪽에서 왼쪽 (Vector3.left)
- Point 7~8: 왼쪽에서 오른쪽 (Vector3.right)
- Point 9, 11: 대각선 좌→우하단
- Point 10, 12: 대각선 우→좌하단

#### 2. 적 이동 / HP / 애니메이션 — EnemyController.cs
**경로:** `Assets/2.SDH/Scripts/EnemyController.cs`

- 코루틴 Move(): 지정 방향으로 이동 → 화면 밖 이탈 시 풀 반환
- onDie (Action\<Vector3\>) 대리자로 사망 위치 전달
- HitRoutine() 코루틴: Hit 애니 → Idle 자동 복귀
- isDead 플래그: 다중 총알 동시 피격 시 중복 사망 방지

```csharp
public Action<Vector3> onDie;
// 사망 시
onDie?.Invoke(transform.position);
```

#### 3. 적 총알 발사 — EnemyBullit.cs / EnemyBullitController.cs
**경로:** `Assets/2.SDH/Scripts/`

- OnEnable()에서 FireRoutine() 코루틴 시작 → 풀 재활성화 시 자동 재시작
- 2초마다 Player 위치로 조준 발사
- EnemyBulletPool(싱글톤) 연동으로 총알 오브젝트 풀링

#### 4. 적 총알 풀 — EnemyBulletPool.cs
**경로:** `Assets/2.SDH/Scripts/EnemyBulletPool.cs`

- 싱글톤, Queue 기반 풀링
- Get() / Return() API

#### 5. 아이템 생성 — ItemManager3.cs / Item3.cs
**경로:** `Assets/2.SDH/Scripts/`

- ItemManager3: 싱글톤, Dictionary 기반 풀링 (coin / power / boom 3종)
- CreateItem(Vector3): 확률 분기 후 풀에서 아이템 꺼내 배치
- Item3: OnEnable()에서 Move() 코루틴 시작 → 낙하 → 풀 반환

**드랍 확률:** None 30% / Coin 30% / Power 20% / Boom 20%

#### 6. Boom 이펙트 — BoomEffect.cs
**경로:** `Assets/2.SDH/Scripts/BoomEffect.cs`

- static event Action OnBoom 발행
- EnemyController / EnemyBullitController가 OnEnable/OnDisable에서 구독·해제
- Boom 발동 시 활성화된 모든 적·적 총알 즉시 처리

```csharp
// BoomEffect.cs
public static event Action OnBoom;
OnBoom?.Invoke();

// EnemyController.cs
void OnEnable()  { BoomEffect.OnBoom += BoomKill; }
void OnDisable() { BoomEffect.OnBoom -= BoomKill; }
```

## 신승환 (SSH) 작업 내역

### 완성된 시스템

#### 1. Player 이동 / 발사 / 피격 — Player.cs
**경로:** `Assets/1.SSH/2.Scripts/Player.cs`

- 이동: GetAxisRaw + Viewport 클램프 (화면 밖 이탈 방지)
- 총알 발사: 좌클릭 홀드, fireInterval(0.2초) 간격
- 3단계 파워업: power=1(정면1) / power=2(양옆2) / power=3(정면+양옆3)
- 싱글톤(Player.Instance)으로 외부 참조 지원
- PlayerBulletPool(싱글톤) 연동으로 총알 풀링

**피격 / 리스폰 흐름:**
```
적 총알 충돌 → TakeDamage()
  → hp > 0: InvincibleRoutine(1.5초 무적)
  → hp = 0: hp 리셋, power=1, RespawnRoutine()
              SpriteRenderer 숨김(1초) → 시작 위치 이동
              SpriteRenderer 표시 → 2초 무적 후 해제
```

**Boom 우클릭:**
```
GetMouseButtonUp(1) → UIManager.UseBoom() → Instantiate(Boom_Boom.prefab)
```

**아이템 획득 (OnTriggerEnter2D):**
```
Coin  → AddScore(100)
Power → PowerUp() + AddScore(200)
Boom  → AddBoom() + AddScore(300)
→ ItemManager3.ReturnItem()으로 풀 반환
```

#### 2. 플레이어 총알 — PlayerBullet.cs / PlayerBulletPool.cs
**경로:** `Assets/1.SSH/2.Scripts/`

- PlayerBulletPool: 싱글톤, Dictionary 기반 2종(side/center) 풀 관리
- PlayerBullet: 위로 이동 → 화면 밖 풀 반환 / 적 충돌 시 TakeDamage(damage) 호출

#### 3. 배경 무한 스크롤 — BackgroundScroller.cs
**경로:** `Assets/1.SSH/2.Scripts/BackgroundScroller.cs`

- 3개 Background 프리팹(y=0/12/24)이 루프 스크롤
- SSH 씬 및 SDH 씬 모두 배치 완료

#### 4. UI 관리 — UIManager.cs
**경로:** `Assets/1.SSH/2.Scripts/UIManager.cs`

- 싱글톤 (UIManager.Instance)
- 목숨 UI (Life_1~3 Image alpha 제어)
- Boom UI (Boom_1~3 Image alpha 제어)
- 점수 텍스트 (TextMeshPro, #,##0 포맷)
- 게임오버 패널 + 최종점수 표시
- Retry 버튼 → 현재 씬 재로드

**공개 API:**
```csharp
UIManager.Instance.AddScore(int amount)
UIManager.Instance.DecreaseLife()   // bool 반환, true면 게임오버
UIManager.Instance.AddBoom()
UIManager.Instance.UseBoom()        // bool 반환
```
