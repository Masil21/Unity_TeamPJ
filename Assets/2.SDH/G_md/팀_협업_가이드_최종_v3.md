## 팀원 정보
- **신동혁 (SDH 브랜치)** — Enemy, Item, UI배치 파트
- **신승환 (SSH 브랜치)** — Player, 총알, 배경, UI관리 파트

## 프로젝트 개요
- Unity 2D 세로형 슈팅 게임
- 해상도: 1920x1080 Portrait
- 프레임: 60fps 고정
- 코드 패턴: 싱글톤, 대리자(Action), 코루틴 활용

## Git 브랜치 구조
```
main (통합용, merge 대상)
 ├─ SSH (신승환 브랜치)
 └─ SDH (신동혁 브랜치)
```
- 각자 브랜치에서 작업 → main에 merge
- merge 후 상대방은 main에서 pull 받아서 최신 상태 유지
- **같은 씬 파일(.unity)은 동시에 수정하지 않기**
- **UIManager.cs는 신승환이 전부 관리** (한 사람이 관리해야 충돌 방지)

### 사용 패턴
- **싱글톤:** `ItemManager3.Instance.CreateItem(pos)`
- **대리자:** EnemyController 안에 `Action<Vector3> onDie` → SpawnPoint에서 생성 시 등록
- **코루틴:** Item3.Move()로 아이템 낙하, Boom 2초 지속 효과
- **Invoke:** hit 스프라이트 복귀

### 대리자 흐름 (아이템 드랍)
```
SpawnPoint → 적 Instantiate
           → ec.onDie += ItemManager3.Instance.CreateItem  // 등록

적 피격 → hp <= 0 → onDie?.Invoke(transform.position)    // 실행
                  → ItemManager3.CreateItem(pos)
                  → 코루틴으로 아이템 낙하
```

### 적 SpawnPoint 배치
- Point 0~4: 위쪽에서 아래로 (Vector3.down)
- Point 5~6: 오른쪽에서 왼쪽으로 (Vector3.left)
- Point 7~8: 왼쪽에서 오른쪽으로 (Vector3.right)
- Point 9→10: 대각선 (방향벡터 계산)
- Point 11→12: 대각선 (방향벡터 계산)
- Point 10, 12는 목표 방향용 (적 생성 안 함)

### 아이템 드랍 확률
- None: 30% (아무것도 안 나옴)
- Coin: 30%
- Power: 20%
- Boom: 20%

### Boom 이펙트/범위 파괴 (신동혁 관리)
- SSH(Player.cs)에서 우클릭 감지 → `UIManager.Instance.UseBoom()` 호출
- SDH는 해당 신호를 받아 Boom_Animation 프리팹 생성
- 2초간 코루틴으로 매 프레임 적/적총알 파괴
- 적 파괴 시 TakeDamage(9999)로 onDie 호출 → 아이템 드랍

```csharp
// 1. 필드 추가
private bool isInvincible = false;

// 2. 적 총알에 맞았을 때 — EnemyBullitController.cs에서 호출
public void TakeDamage()
{
    if (isInvincible) return;
    power = 1;
    UIManager.Instance.HandlePlayerHit(gameObject, transform.position, 1f);
}

// 3. Power 아이템 획득 시 — Player OnTriggerEnter2D에서 호출
public void PowerUp()
{
    if (power < 3) power++;
}

// 4. 리스폰 무적 설정 — UIManager.RespawnPlayer에서 호출
public void SetInvincible(bool value)
{
    isInvincible = value;
}

// 5. Boom 아이템 획득 시 — Player OnTriggerEnter2D에서 호출
public void AddBoom()
{
    UIManager.Instance.AddBoom();
}

// 6. 아이템 획득 처리 — 전체 없음
void OnTriggerEnter2D(Collider2D other)
{
    Item3 item = other.GetComponent<Item3>();
    if (item == null) return;

    switch (item.itemType)
    {
        case Item3.ItemType.Coin:
            UIManager.Instance.AddScore(100);
            break;
        case Item3.ItemType.Power:
            PowerUp();
            UIManager.Instance.AddScore(200);
            break;
        case Item3.ItemType.Boom:
            AddBoom();
            UIManager.Instance.AddScore(300);
            break;
    }
    Destroy(other.gameObject);
}
```

## 연결 포인트 (인터페이스 약속)

### 신동혁 → 신승환 (동혁이 만들고 승환이 사용)
| 제공 | 내용 |
|------|------|
| Item3.cs | `GetComponent<Item3>()` → `item3.itemType`으로 판별 |
| Item3.ItemType | `enum { None = -1, Coin, Boom, Power }` |

### 신승환 → 신동혁 (승환이 만들고 동혁이 사용)
| 제공 | 내용 |
|------|------|
| Player.TakeDamage() | 적 총알 충돌 시 동혁이 호출 |
| Player.PowerUp() | Power 아이템 획득 시 동혁이 호출 |
| Player.SetInvincible(bool) | 리스폰 무적 시 UIManager 내부에서 호출 |
| Player.AddBoom() | Boom 아이템 획득 시 동혁이 호출 |
| UIManager.Instance.AddScore(int) | 점수 추가 시 동혁이 호출 |
| UIManager.Instance.UseBoom() | Boom 사용 판단 (Player.cs 우클릭에서 이미 호출 중) |

### 신동혁이 신승환 코드를 호출하는 코드 예시

```csharp
if (player != null)
{
    player.TakeDamage();
    Destroy(gameObject);
}

// 아이템 드랍 위치에서 점수 추가 (Player.cs OnTriggerEnter2D)

UIManager.Instance.AddScore(200); // Power
UIManager.Instance.AddScore(300); // Boom
```
