### 4. EnemyController.cs 생성
**경로:** `Assets/2.SDH/Scripts/EnemyController.cs`

- 아래로 이동 (speed=1f), y≤-5.5f 이하 떨어지면 Destroy
- `OnTriggerEnter2D` → "PlayerBullet" 태그 감지 → TakeDamage(5)
- hp=16, TakeDamage() → hp≤0이면 Destroy
- Hit 애니메이션 재생 후 자동 Idle 복귀 (코루틴)
- `if (this == null) yield break;` null 체크 적용

```csharp
// 핵심 구조
public float speed = 1f;
public int hp = 16;

public void TakeDamage(int damage) {
    hp -= damage;
    if (hp <= 0) { Destroy(gameObject); return; }
    StartCoroutine(HitRoutine()); // Hit 애니 → Idle 복귀
}
```

## 충돌 처리 흐름

```
SSH 총알(PlayerBullet 태그) → Enemy CapsuleCollider2D 진입
  → EnemyController.OnTriggerEnter2D 발동
  → TakeDamage(5) 호출
  → hp 감소
    ├─ hp > 0 → HitRoutine() → Hit 애니 → Idle 복귀
    └─ hp ≤ 0 → Destroy(gameObject)
```
